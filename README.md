# 📖 玲华阅读

> 一款简洁优雅的 Android 阅读器，专为爱读书的你打造 ✨

---

## 🔥 为什么选择玲华阅读？

还在为找不到好用的阅读 App 发愁吗？市面上的阅读器要么广告满天飞，要么功能臃肿用不上。**玲华阅读**就是那个「刚刚好」的选择——**干净、纯粹、只为阅读**。

---

## 💡 核心亮点

### 📚 多格式支持
TXT / EPUB / PDF 通通拿下，本地导入一键搞定

### 🔍 在线搜索 + 一键下载
内置书源，搜索即下载，**边下边看**不用等，下载进度实时可见

### 🎨 高颜值阅读界面
- 多种阅读主题随心切换（浅色 / 深色 / 护眼 / 自定义）
- 字号、行距自由调节
- 滚动、翻页、仿真、覆盖四种翻页模式任选

### 🌙 深色模式
跟随系统自动切换，夜间阅读不伤眼

### 🎯 Material You 动态配色
Android 12+ 自动适配壁纸配色，和你的手机浑然一体

### 📊 阅读统计
记录你的每一分钟阅读时光，日历热图一目了然

### ☁️ WebDAV 同步
阅读进度、书架数据云端备份，换机无忧

### 🔄 应用内更新
发现新版本？一键下载安装，进度条清晰可见，更新体验丝滑

## 📱 功能一览

| 功能 | 描述 |
|------|------|
| 本地导入 | 支持 TXT、EPUB、PDF |
| 在线搜索 | 内置书源，搜书即下载 |
| 边下边看 | 下载未完成也能开始阅读 |
| 智能章节 | 自动识别章节目录 |

| 书架管理 | 排序、搜索、长按删除 |
| 阅读统计 | 时长记录 + 日历热图 |
| 数据同步 | WebDAV 云端备份恢复 |
| 应用更新 | 应用内下载安装 |

## 🛠️ 技术栈

给开发者小伙伴看的 👇

- **语言**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **数据库**: Room
- **偏好存储**: DataStore
- **网络**: OkHttp + Jsoup
- **图片加载**: Coil
- **CI/CD**: GitHub Actions 自动构建发布
